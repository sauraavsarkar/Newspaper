The 3-Phase Attack Plan

01
Scaffold & Auth
Run Laravel installer, configure MySQL, add Spatie roles. Get login + register + email verify working first. Everything else depends on this.
Day 1 · Hours 1–4
02
Core Article CRUD
Articles model with categories, tags, slug, status (draft/published/scheduled). CKEditor in the form. Livewire table for listing. Policies to guard roles.
Day 1 · Hours 5–10
03
Editorial Workflow
Status machine: draft → pending_review → approved → published. Rejection notes. Activity log. This is what makes interviewers lean forward.
Day 1 · Hours 11–14
04
Breaking News & Real-Time
Laravel Broadcasting + Pusher. Breaking news flag on article. Live ticker on front-end via Alpine.js Echo listener. Works without WebSockets using polling fallback.
Day 2 · Hours 1–5
05
Analytics Dashboard
article_views table with IP + timestamp. Trending score = views + comments weighted by time decay. Chart.js graphs for views-per-day and top categories.
Day 2 · Hours 6–10
06
Polish & Ship
Responsive Tailwind layout, seed 50 fake articles (Faker), write README with ER diagram + demo creds, push to GitHub, deploy to Railway or Render.
Final 4 Hours

Your First Terminal Session (Do This Now)

# 1. Create the project composer create-project laravel/laravel newspaper-cms cd newspaper-cms # 2. Install core packages composer require spatie/laravel-permission composer require livewire/livewire # 3. Publish + migrate php artisan vendor:publish --provider="Spatie\Permission\PermissionServiceProvider" php artisan migrate # 4. Install front-end npm install -D tailwindcss @tailwindcss/typography alpinejs npx tailwindcss init # 5. Make your first models php artisan make:model Article -mfsc php artisan make:model Category -mf php artisan make:model ArticleView -m

Tech Stack at a Glance

LayerToolsWhy
Auth & Roles Laravel Breeze · Spatie Permission Fastest gate/policy setup; roles stored in DB
Frontend Livewire 3 · Alpine.js · Tailwind CSS Full reactivity without writing an API; Alpine handles small interactions
Rich Text CKEditor 5 (CDN) Paste from Word, image upload, clean HTML output
Real-Time Pusher · Laravel Echo Free tier is enough for demos; fallback to polling

Speed Tips That Save Hours

Use make:model -mfsc
One command generates the Model, Migration, Factory, Seeder, and Controller together. Never generate them separately.
Skip building a REST API
Livewire replaces the need for a separate API for your own frontend. Only build REST if you plan to show a mobile app too.
Seed aggressively early
Run php artisan db:seed after every major migration. You need realistic data to see if your dashboards look good.
Use Spatie packages
Permission, MediaLibrary, ActivityLog — all production-grade, all on your CV. Beats reinventing the wheel in 28 hours.
